# Pong Game (OOP Implementation)

A classic two-player Pong game built with Python and Pygame, demonstrating core Object-Oriented Programming (OOP) principles.

## Features
- Player vs Player controls (W/S and Up/Down arrows)
- Score tracking
- Clean object structure: `Paddle`, `Ball`, and game loop

## Requirements
- Python 3.x
- Pygame
